# Traffic-light-Simulation

A visually engaging traffic light simulation built with vanilla HTML, CSS, and JavaScript.


## ✨ Key Features

-   **Realistic Animations:** Smooth light transitions with CSS glow effects.
-   **Dynamic Timer:** A digital LCD-style countdown timer for each light state.
-   **Pure Front-End:** Built with zero dependencies, just core web technologies.
-   **Responsive Design:** Looks great on both desktop and mobile screens.

## 🛠️ Tech Stack

-   **HTML**
-   **CSS** (Flexbox, Gradients, Box-Shadows)
-   **JavaScript** (DOM Manipulation, `setInterval`)
